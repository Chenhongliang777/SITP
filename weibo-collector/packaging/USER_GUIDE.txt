CSL Sentinel 足球舆情监测系统 — 使用说明（Windows 安装版）
================================================================

一、安装与启动
  1. 运行 CSLSentinel_Setup.exe 完成安装（或使用绿色版文件夹内的 CSLSentinel.exe）。
  2. 双击 CSLSentinel.exe 打开桌面程序，无需安装 Python。
  3. 首次启动会自动创建 data、reports、logs 目录及 .env 配置文件。

二、首次配置（约 3 分钟）
  1. 打开「设置」页，填写大模型 API Key（默认 DeepSeek 预设即可）。
  2. 打开「微博登录」页，点击「打开微博登录窗口」，浏览器扫码后点「确认已登录并保存」。
  3. 以上完成后即可在「监测任务」页开始监测。

三、手机访问（可选）
  1. 电脑与手机连接同一 WiFi。
  2. 在 GUI「手机访问」页开启局域网服务，用手机浏览器打开显示的地址。
  3. 将 PC 上显示的「访问令牌」填入手机页面并保存。
  4. 微博登录仍在电脑上完成；手机负责下任务、看进度与报告。

四、监测任务
  - 填写关键词、日期范围、目标条数。
  - 「高效模式」可显著加快速度（少调用大模型，质量略降）。
  - 完成后会自动打开 HTML 报告，也可在 reports 文件夹中查看历史报告。

五、常见问题
  - 防火墙：若手机无法访问，请在 Windows 防火墙中允许 CSLSentinel 专用网络访问（端口 8765）。
  - 杀毒软件：本程序内置 Chromium 与 AI 模型，体积较大，部分杀毒可能误报，请添加信任。
  - API Key：未配置时无法开始监测，请在「设置」页保存。
  - 磁盘空间：建议预留 8GB 以上（含模型与浏览器缓存）。

六、目录说明（安装目录旁）
  data/      中间数据与微博登录态
  reports/   HTML 研判报告
  logs/      采集日志
  vendor/    内置模型与 Chromium（勿删）
  .env       API 与模型配置（可用记事本编辑，推荐用 GUI 设置页）

技术支持：请参阅项目 README 或联系维护者。
